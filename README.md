# Personal Portfolio Website

## Overview

A modern, fully responsive portfolio website built using semantic HTML5, advanced CSS3, and JavaScript. This project demonstrates accessibility best practices, responsive design techniques, and modern front-end development principles.

## Features

* Semantic HTML5 structure
* CSS Grid and Flexbox layouts
* Mobile-first responsive design
* Light/Dark mode theme toggle
* WCAG accessibility support
* SEO-friendly meta tags
* Keyboard navigation support
* Accessible contact form
* GitHub Pages ready

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* CSS Grid
* Flexbox
* Responsive Web Design

The website adapts to:

* Mobile Devices (320px+)
* Tablets (768px+)
* Desktop Screens (1024px+)
* Large Displays (1400px+)

## Dark Mode

The website includes a theme toggle that:

* Switches between light and dark themes
* Saves user preference using localStorage
* Restores theme on page reload

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio-website.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

## Deployment with GitHub Pages

1. Push the project to GitHub.
2. Open Repository Settings.
3. Select **Pages**.
4. Choose:

   * Source: Deploy from a branch
   * Branch: main
   * Folder: /(root)
5. Save changes.

Your site will be available at:

```text
https://your-username.github.io/portfolio-website/
```
